# 雅丽洁防晒乳 AIGC 短片 · 本地一键生成说明

沙盒环境连不上 `api.x.ai`（被网络策略拦截），所以真实生成要在你本地跑。
脚本 `generate_video.py` 已写好，全流程自动：调用 xAI 生成 7 个镜头 → 下载 → ffmpeg 拼成 33 秒竖屏成片。

## 一、准备环境（只需一次）

```bash
# 1. Python 依赖
pip install requests

# 2. ffmpeg（拼接用）
#   macOS:   brew install ffmpeg
#   Windows: winget install Gyan.FFmpeg   或去 ffmpeg.org 下载
#   Linux:   sudo apt install ffmpeg
ffmpeg -version   # 能打印版本就说明装好了
```

## 二、放两样东西到脚本同目录

1. `generate_video.py`（本脚本）
2. `product.jpg` —— 你的产品白底图（镜头 2、3 会用它做"图生视频"）

## 三、设置 API Key 并运行

```bash
export XAI_API_KEY="xai-你的key"          # Windows PowerShell: $env:XAI_API_KEY="xai-..."
python3 generate_video.py
```

跑完会得到：
- `shots/`  —— 每个镜头的 mp4
- `final_output.mp4` —— 拼好的完整成片（已去音轨，方便你导入剪映统一配乐+加字幕）

## 四、常用参数

```bash
# 断点续跑：跳过已生成的镜头
python3 generate_video.py --skip-existing

# 只重跑某几个镜头（比如第 1、3、6 个不满意）
python3 generate_video.py --only 1,3,6

# 换用 1.5 模型
python3 generate_video.py --model grok-imagine-video-1.5
```

## 五、说明

- **镜头 5（时间翻牌 7:50→7:52）和收尾 Logo 定版** 按脚本原意本就该用剪映模板做，
  不适合 AI 生成，脚本里直接用 ffmpeg 生成了文字卡片，保证一条命令跑通。
  如果你想在剪映里做更精致的翻牌动效，把这两段换掉即可。
- **计费**：每个镜头都会消耗你 xAI 账户额度，7 个镜头里有 6 个走 API（镜头5、收尾不走）。
  建议先 `--only 1` 试跑一个镜头，确认效果和扣费正常，再整体跑。
- **字段兼容**：脚本对返回字段（request_id / status / url）做了多种命名兼容；
  万一 xAI 接口有细微调整报错，错误信息会打印完整返回，照着调一下 body 字段即可。
- **安全**：你之前在对话里贴过的那把 key 建议用完去 console.x.ai 撤销重建，避免泄露。
